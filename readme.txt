CRMPRO Importer for Wordpress lets you import, merge and sync users straight into the CRM.


== Description ==

The CRMPRO Importer plugin for Wordpress enables you to import your WP users into the CRM.  The Importer installs as a Tool and can be used to import existing users (subscribers, contributors, authors, editors, admins) to the CRM.  

All you need to do is set up the plugin with the web services key in CRMPRO.  Simply copy and paste in your Web Services Key and the User ID for the CRMPRO user account.  After this you can access the Tools section and import existing users.

CRMPRO Importer requires an account at CRMPRO.com
